# Detmud dotfiles

## Installation

```bash
git clone git://github.com/detmud/dotfiles.git
cd ~/dotfiles
chmod +x makesymlinks.sh
./makesymlinks.sh
```

## Installation on a new Mac

* Install Brew 
```bash
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
```

* Install ZSH
```bash
brew install zsh zsh-completions
```
Follow Instructions !

* Install VIM
```bash
brew install vim
```
